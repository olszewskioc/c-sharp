# Commands
- `dotnet new console -n <nome-arquivo>` (Create an new console Application)
- `dotnet run` (Running Main)
- `dotnet sln <nome>.sln add <DIR\FILE>.csproj>` (Add csproj in sln to debug)
- `dotnet clean`
- `dotnet build`
